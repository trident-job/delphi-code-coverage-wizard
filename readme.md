    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-delphicodecoveragewizard" class="anchor" href="#delphicodecoveragewizard" aria-hidden="true"><span class="octicon octicon-link"></span></a>DelphiCodeCoverageWizard</h1>

<h2>
<a id="user-content-introduction" class="anchor" href="#introduction" aria-hidden="true"><span class="octicon octicon-link"></span></a>Introduction</h2>

<p><strong>DelphiCodeCoverageWizard</strong> is a simple wizard to setup and build scripts for <a href="http://code.google.com/p/delphi-code-coverage">DelphiCodeCoverage</a> - a simple Code Coverage command-line tool for <strong>Delphi</strong>.</p>

<h2>
<a id="user-content-preconditions" class="anchor" href="#preconditions" aria-hidden="true"><span class="octicon octicon-link"></span></a>Preconditions</h2>

<ul class="task-list">
<li>A Delphi project executable (.exe)</li>
<li>A detailed mapping file for this project (.map).
<em>Note : The wizard has it own copy of DelphiCodeCoverage.exe</em>
</li>
</ul>

<h2>
<a id="user-content-what-help-does-it-provide" class="anchor" href="#what-help-does-it-provide" aria-hidden="true"><span class="octicon octicon-link"></span></a>What help does it provide</h2>

<p>DelphiCodeCoverageWizard will generate the command-line and additional files to run the coverage for this project.</p>

<p>You can also choose which unit to cover.</p>

<h2>
<a id="user-content-usage" class="anchor" href="#usage" aria-hidden="true"><span class="octicon octicon-link"></span></a>Usage</h2>

<p>Download the application.
Run it and follow the wizard !</p>

<h2>
<a id="user-content-output" class="anchor" href="#output" aria-hidden="true"><span class="octicon octicon-link"></span></a>Output</h2>

<h3>
<a id="user-content-script" class="anchor" href="#script" aria-hidden="true"><span class="octicon octicon-link"></span></a>Script</h3>

<p>A batch file (.bat) is created for a easier one-click coverage execution.</p>

<p>Two listing files (.lst) containing units list and units folder list will be created.</p>

<h3>
<a id="user-content-coverage-output" class="anchor" href="#coverage-output" aria-hidden="true"><span class="octicon octicon-link"></span></a>Coverage output</h3>

<p>The coverage report is from DelphiCodeCoverage. For further information, see the project page.</p>

<h2>
<a id="user-content-credit" class="anchor" href="#credit" aria-hidden="true"><span class="octicon octicon-link"></span></a>Credit</h2>

<p>A really big thanks to <a href="http://christerblog.wordpress.com/">Christer Fahlgren</a> for his delphi-code-coverage release.</p>

<p>Find more here : <a href="http://code.google.com/p/delphi-code-coverage/">http://code.google.com/p/delphi-code-coverage/</a></p>
